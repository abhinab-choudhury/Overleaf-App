# Overleaf Desktop

Overleaf Desktop is a lightweight Electron-based desktop wrapper for Overleaf, providing a native app-like experience on your system.

## Installation

### Windows

1. Download the `.exe` file from **Releases**
2. Double-click and follow the installer steps

### Debian / Ubuntu

```bash
sudo dpkg -i overleaf-desktop_*.deb
```

### AppImage (Portable)

```bash
chmod +x Overleaf-Desktop-*.AppImage
./Overleaf-Desktop-*.AppImage
```

## Notes

* This app is an **unofficial desktop wrapper** for Overleaf
* Internet connection is required
* No Overleaf credentials are stored locally
